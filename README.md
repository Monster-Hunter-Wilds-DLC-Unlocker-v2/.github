## 🔧 Troubleshooting

| Problem                        | Solution                                                                 |
|--------------------------------|--------------------------------------------------------------------------|
| DLCs not showing in-game        | Run **CreamAPI as Administrator** and restart Steam.                     |
| Game crashes after launch       | Use the **Uninstall** option in CreamAPI and reinstall the unlockers.    |
| Missing DLL errors              | Reinstall **Koaloader** via CreamInstaller’s context menu.               |
| Unlocker still doesn’t work     | Re-run **CreamAPI** and refresh Steam appinfo for **Monster Hunter: Wilds**. |
